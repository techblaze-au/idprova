# Decision Gates

> Review these monthly. They tell you when to push harder and when to pivot.

---

## Gate 1 — Month 3 (June 2026)

**Question:** Is there developer interest?

| Criteria | Target | How to Measure |
|----------|--------|----------------|
| GitHub stars | 500+ | GitHub repo page |
| Framework integration | 1+ (LangChain, MCP, or CrewAI) | Published tutorial + working code |
| HN front page | Yes | HN analytics / post history |
| External contributors | 3+ PRs merged | GitHub PR history |
| crates.io downloads | 1000+ total | crates.io stats |

**If YES:** Double down on developer content, start enterprise outreach.
**If NO:** Reassess messaging. Try different angle (security-first vs dev-first). Consider pivoting the narrative.

---

## Gate 2 — Month 6 (September 2026)

**Question:** Can this generate revenue?

| Criteria | Target | How to Measure |
|----------|--------|----------------|
| GitHub stars | 200+ (cumulative) | GitHub |
| Enterprise PoC | 1+ started | Direct conversations |
| Paid consulting | 1+ gig completed | Invoice records |
| Conference talk | 1+ delivered | Event confirmation |

**If YES:** Hire DevRel contractor (2 days/week), scope IDProva Cloud SaaS.
**If NO:** Reduce to maintenance mode. Focus consulting revenue on DISP/E8 workshops instead. IDProva becomes a credential, not a business.

---

## Gate 3 — Month 12 (March 2027)

**Question:** Is this a business or a side project?

| Criteria | Target | How to Measure |
|----------|--------|----------------|
| Monthly recurring revenue | $10K+ MRR | Stripe/invoicing |
| Enterprise contracts | 3+ active | Contract records |
| Community size | 500+ Discord/GitHub | Platform analytics |
| Growth trajectory | Month-over-month increase | Trend analysis |

**If YES:** Decide — raise seed funding OR stay bootstrapped with profitable growth.
**If NO:** Sunset active development. Keep as open-source credential on resume. Focus entirely on consulting revenue streams (DISP, E8, compliance).
